Class: contextmenu

右键菜单

Author: putaoshu@126.com.
View source: contextmenu.js.

Arguments:

vui.contextmenu (obj, options)

  1. obj - (Object) 主对象
  2. options - (Object) 组件配置
  3. options.menu - (Object) null 菜单主体
  4. options.zIndex - (Nummber) 2 菜单z-index
  5. options.autoShow - (Boolean) true 自适应:即在主体元素有被遮挡时,把主体转移到上面
  6. options.preventContextmenu - (Boolean) true 阻止系统默认右键菜单
  7. options.diff - (Number) 2 当前位置与菜单展示位置的差值
  8. options.callback - (Function) 回调函数

Example: 

vui.contextmenu($('#test'),{
	menu:$('#contextmenu')
})
